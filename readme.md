Project Flaming Octo Dangerzone
===============================

A Twitter clone, written in Java, for a CPSC 330 project.

Features
--------

  - user accounts
    - profiles
    - bio information
  - public timeline
  - personalized timeline for logged-in users (ability to follow other users)
  - persistent information (SQLite database)
  - hashtags
  - @user messages
  - GUI
